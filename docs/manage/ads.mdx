# Manage Ads

## Overview

In this section, you can create, update and manage all ads available on the website.

To view an overview of the table list and common content management opearations such as creating, searching, and more, please refer to [this section](../getting-started/overview#collections).

---

## Fields

![Ad Fields](https://wvn.hcm.ss.bfcplatform.vn/uploads/ad-fields-v2.png)

:::danger NOTE
If there is a `*` behide the field name, that field is **required**.
:::

Next, we will go into detail about each field:

#### 1. Campaign

The campaign represents an advertising campaign

#### 2. Target Url

When users click the advertisement, the system will redirect them to the **Target Url**.

#### 3. Media

The media can be either an **Image** or a **Video**.

:::danger NOTE
Base on **Position Field (4)**, you must choose the approriate Media format.

- If the position is **Vertical**, you must select **Vertical Media**

- If the position is **Horizontal**, you must select **Horizontal Media**
  :::

Example of ad media in both cases:

#### Vertical Media

![Ad vertical media](https://wvn.hcm.ss.bfcplatform.vn/uploads/ad-vertical-media.png)

#### Horizontal Media

![Ad horizontal media](https://wvn.hcm.ss.bfcplatform.vn/uploads/ad-horizontal-media.png)

#### 4. Position

The position can be either **Vertical** nor **Horizontal**.

Use the **Horizontal** option to display ads at the **top of the website** or on the **Home Page**

Use the **Vertical** option to display ads in side positions, such as in block **Category Spotlight** or page **Article Detail**.

:::danger NOTE
You must select this field correctly, as it determines where your ads will appear in different components.

:::

**Example:**

In the block **Ad Banner**, only list of **Horizontal Ad** is displayed.

![Ad position example horizontal](https://wvn.hcm.ss.bfcplatform.vn/uploads/ad-position-exp-1.png)

In the block **Category Spotlight**, only list of **Vertical Ad** is displayed.

![Ad position example vertical](https://wvn.hcm.ss.bfcplatform.vn/uploads/ad-position-exp-2.png)

#### 5. Start Date

Set the start date of the advertisement here.

#### 6. End Date

Set the end date of the advertisement here.

:::warning WARNING
The **end date** must be later than the **start date**.

If the dates are set incorrectly, the system will return an error.
:::
