# Manage Users

## Overview

In this section, you can create, update and manage all ads available on the website.

To view an overview of the table list and common content management opearations such as creating, searching, and more, please refer to [this section](../getting-started/overview#collections).

---

## Fields

![User overview](https://wvn.hcm.ss.bfcplatform.vn/uploads/user-fields.png)

:::danger NOTE
If there is a `*` behide the field name, that field is **required**.
:::

Next, we will go into detail about each field:

#### 1. Email

You can set the user's email here.

:::danger NOTE
Email must be **unique**. If you create an email that already exists, the system will return an error.
:::

#### 2. New Password

There is a user's password.

#### 3. Confirm Password

There is a confirmination password. You must enter the same value as the new password. If not, the system will return an error.

#### 4. Name

There is the user's display name.

#### 5. Roles

You can assign roles to the user here. There are some available roles in website, such as: **ADMIN**, **SUPERVISOR**, **EDITOR**.
