# Manage Categories

## Overview

In this section, you can create, update and manage all categories available on the website.

To view an overview of the table list and common content management opearations such as creating, searching, and more, please refer to [this section](../getting-started/overview#collections).

---

## Fields

![Category overview](https://wvn.hcm.ss.bfcplatform.vn/uploads/categories.png)

:::danger NOTE
If there is a `*` behide the field name, that field is **required**.
:::

Next, we will go into detail about each field:

#### 1. Name

The name represents the display name of category.

#### 2. Description

The description is a short summary of category.

#### 3. Slug

The slug is the URL-friendly identifier of category.

In the website, the slug is used when users search for posts by category.

![Category search](https://wvn.hcm.ss.bfcplatform.vn/uploads/category-search.png)

:::tip TIP
The slug can be **auto-generated** from **name**. So, when creating a new category, you can leave the slug field empty.
:::

:::danger NOTE
Slug must be **unique**. If you create a slug that already exists, the system will return an error.
:::
