# Introduction on Category Spotlight

The `Category Spotlight` section highlight content from a specific category.

It displays a main post and additional related posts from the same category.

:::tip
This section requires the user to enter a category, so it only fetches the latest posts from the selected category.
:::

## Basic Usage

To use the `Category Spotlight` section: - Click on `Add Section` in the CMS. - Choose `Category Spotlight` from the section list. - Fill in the setting as needed.

![Category Spotlight Added](https://wvn.hcm.ss.bfcplatform.vn/uploads/add-category-spotlight-section.png)

## Detailed Field Explanation

### 1. Category <ColorBox type="text" color="red">\*</ColorBox>

> Choose the category to display posts.

- This field is required.
- You can select a category from the **Categories Collection**.
- The main post and related posts will all be pulled from this category.

### 2. Limit

> Set how many posts to display .

- You can choose between 1 - 5 posts(default auto select 4).
- The firse post will be show as the main story.
- Remaining posts will appear as related posts.

### 3. Layout Options(Grouped under "Option")

> Control how the spotlight section appears on the website.

#### `Full width` : Main post image spans the full page width.

![preview render ui full width](https://wvn.hcm.ss.bfcplatform.vn/uploads/category-spotlight-fullwidth.png)

#### `Container Fixed Width` : main post image spans the width of the container.

![preview render ui container ](https://wvn.hcm.ss.bfcplatform.vn/uploads/category-spotlight-container.png)

#### `Flex columns` : Content is displayed in a 3 columns layout.

You can fill in **Custom Meta Title & Description(optional)** - **Use gray background(optional)** when using `Flex columns` layout.

This is the preview of `Flex columns` layout on homePage.

![preview render ui flex columns](https://wvn.hcm.ss.bfcplatform.vn/uploads/category-spotlight-columns.png)

:::warning
When selecting the `Flex columns` layout, the **Enable Ads** option will not be displayed on the website interface, even if **Enable Ads** is turned on or ads items are added to Ad Data.
:::

### 4. Enable Ads

> Toggle to enable ad content in this section.

- If enabled checkbox, the `Ads` group of fields will appear.
- If disable checkbox, the `Ads` group of fields will not appear.
- When this option is disable checkbox, ads will not be shown in the CMS and will not be rendered on the website interface.

### 5. Ads _(Visible only if `Enable Ads` is checked)_

> This group let you add ad content that show next to the featured story.

- **Ad Data**:
  This is a list of Ads you want to show . - Each item lets you select one ad from `Ads` Collection. - The system automatically filter the lisst to only show ads that: - Have `position` set to `vertical`. - Are still valid (between `startAt` and `endAt`). - If no ad selected,nothing will be show on the website.

- **Ad Mode** :
  Control how the selected ads will appear on the website: - **Exclusive**: Show only the **first valid** ad in the list **Ad Data** - **Stacked**: Show **all valid** ads in a vertical layout. - **Random**: Show **one random valid** ad each time the page is loaded. - **Rotate** : Rotate between **ads valid** changing every 30 seconds.

:::tip NOTE

:::
