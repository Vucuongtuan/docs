# Introduction for Figure Media

The `Figure Media` section is designed to manage media content, including images and videos.

This section allows you to add media with captions and other relevant information to enhance the visual appeal of your content.

## Basic Usage

To use the `Figure Media` section, simply click on `Add Section` in the content and select `Figure Media` from the section list.

![Add Figure Media Section](https://wvn.hcm.ss.bfcplatform.vn/uploads/add-figure-media-section.png)

## Detailed Field Explanation

This is an overview of the fields in the `Figure Media` section.

![Figure Media Overview](https://wvn.hcm.ss.bfcplatform.vn/uploads/overview_figure_media.png)

:::warning NOTE
You must fill in all required fields marked with <ColorBox color="red" type="text">(\*)</ColorBox>.
:::

### 1. Layout <ColorBox color="red" type="text">\*</ColorBox>

The `Layout` field allows you to select how the media block will be displayed on the page. You can choose from the following options:

- **Compact**: The media block will have minimal padding and margins.
- **Narrow**: The media block will have slightly more padding than compact, providing a bit more space for the media.
- **Normal**: This is the default layout, providing a balanced amount of padding and margins.
- **Wide**: The media block will have more padding, making it suitable for longer paragraphs or detailed content.
- **Full Width**: The media block will occupy the entire width of the page, making it full-width.

### 2. Media <ColorBox color="red" type="text">\*</ColorBox>

The `Media` field allows you to upload or select an image or video from the `Media Collection`.
You can upload a new file or choose an existing one from the media library.

### 3. Caption

The `Caption` field allows you to add a caption or description for the media content.
This can include text that describes the media, provides context, or adds additional information.
