# Introduction on App Install Prompt


The `App Install Prompt` section is used to promote the WWK mobile for IOS and Android.

It display  a short message,download button link and banner media.


## Basic use

To use the `App Install Prompt` section : 
    - Click on **Add Section** in the CMS .
    - Choose `App Install Prompt` from the section list.
    - Fill in the message,download link and banner media.

![add section](https://wvn.hcm.ss.bfcplatform.vn/uploads/add-app-install.png)

## Detailed Field Explanation

![overview App install Prompt](https://wvn.hcm.ss.bfcplatform.vn/uploads/overview-app-install.png)


### 1. Title <ColorBox type="text" color="red">*</ColorBox>

 A short heading show on the left side of the section.
 Default value : "Tải ứng dụng WWK trên iOS & Android"

### 2. Description <ColorBox type="text" color="red">*</ColorBox>

 A short description show under the title.
 Default value : "Cập nhật liên tục những hành trình mới, địa điểm thú vị, phong cách sống của giới trẻ và những câu chuyện truyền cảm hứng xê dịch."

### 3. Download IOS Link <ColorBox type="text" color="red">*</ColorBox>

Link to the app on the App Store.

### 4. Download Android Link <ColorBox type="text" color="red">*</ColorBox>

Link to the app on the Google Play Store.

### 5. Banner Media <ColorBox type="text" color="red">*</ColorBox>

A banner media show on the right side of the section.
