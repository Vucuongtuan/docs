# Introduction on Job Listing


The `Job Listings` section is used to display open career positions on the website. It supports selecting one or more jobs from the CMS to be displayed in a list format.


## Basic use

To use the `Job Listings` section:

    - Click on **Add Section** in the CMS.
    - Choose `Job Listing` from the section list.
    - Fill in the section title and select available job entries from the careers collection.

![add job](https://wvn.hcm.ss.bfcplatform.vn/uploads/add-job.png)

## Detailed Field Explanation

This is a overview field explanation of the `Job Listings` section.

![job listing](https://wvn.hcm.ss.bfcplatform.vn/uploads/overview-job.png)

### 1. Title

A heading show at the top of this section.

Default value : "Open Positions"

### 2. Jobs
A relationship field that lets you choose one or more job entries from the **`careers` collection**.

Only jobs with published status will appear on the website. For more details about managing jobs, see [Careers Management](/docs/manage/careers) .

## Preview on website
Once this section is configured, the selected jobs will be listed on the page. Each job will show the job type and title, linking to its detailed page.
