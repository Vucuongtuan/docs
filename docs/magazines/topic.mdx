# Topics

## Overview

The **Topics** collection is designed to manage the various topics or categories that can be associated with magazines within the Wowweekend CMS. This allows for better organization and filtering of magazines based on specific themes or subjects.

To view an overview of the table list and common content management opearations such as creating, searching, and more, please refer to [this section](../getting-started/overview#collections).

---

## Fields

Below is a detailed description of the fields in the **Topics** collection, which is used to manage topics for magazines in the Wowweekend CMS.

![Create Topic Overview](https://wvn.hcm.ss.bfcplatform.vn/uploads/topic_create.png)

:::danger NOTE
If there is a `*` behide the field name, that field is **required**.
:::

| STT | Field Name    | Description                                                                                                                       |
| --- | ------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| 1   | `Name`        | The main name of the topic                                                                                                        |
| 2   | `Description` | A brief description of the topic, providing context or details about what the topic covers.                                       |
| 3   | `Slug`        | Unique slug for the topic, used in the URL. This is automatically generated based on the name after saving but can be customized. |
