# Magazines

## Overview

The **Magazine** CMS is designed to manage content for **Wowweekend** brand magazine publications across their website and mobile application.

This collection allows for easy creation, editing, and management of magazine records, ensuring that information is consistently and accurately presented across all platforms.

To view an overview of the table list and common content management opearations such as creating, searching, and more, please refer to [this section](../getting-started/overview#collections).

---

## Fields

Below is a detailed description of the fields within the Magazine collection, used for managing magazine content in the Wowweekend CMS.

![Create Magazine Overview](https://wvn.hcm.ss.bfcplatform.vn/uploads/magazine_create.png)

:::danger NOTE
If there is a `*` behide the field name, that field is **required**.
:::

| STT | Field Name            | Description                                                                                                                                                  |
| --- | --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 1   | `Title`               | The main title of the magazine .                                                                                                                             |
| 2   | `Subtitle`            | A subtitle or short description that complements the main title of this section will appear below the title when displayed on the detailed journal website . |
| 3   | `Description`         | Brief description of the magazine content                                                                                                                    |
| 4   | `Publication Details` | Information about the magazine issue, author, license ,printed at ,price ,publish time,v.v...                                                                |
| 5   | `Slug`                | Unique slug for the magazine issue, used in the URL. This is automatically generated based on the title after saving but can be customized.                  |
| 6   | `ISBN`                | International Standard Book Number for the magazine.                                                                                                         |
| 7   | `Price`               | The price of the magazine.                                                                                                                                   |
| 8   | `Total Pages `        | Total number of pages in the magazine.                                                                                                                       |
| 9   | `Topic`               | The magazine topic or category(selection from relationship with ["Topic"](/docs/magazines/topics) collection). This helps categorize the magazine content.   |
| 10  | `Published At `       | The date and time when the magazine was published.                                                                                                           |
| 11  | `Feature Image`       | The main image representing the magazine.                                                                                                                    |
| 12  | `Sample`              | A Sample file of the magazine ,typically a short version to give users a quick look.                                                                         |
| 13  | `PDF   `              | The full PDF file of the magazine, allowing users to download or view in detail.                                                                             |
| 14  | `ePUB  `              | The complated magazine in ePUB format, suitable for e-readers and mobile devices.                                                                            |
| 15  | `SEO   `              | SEO settings for the magazine, including meta title, description, and keywords to improve search engine visibility.                                          |
